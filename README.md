TeamFlow is a full-stack project & task management system
This repository contains the backend API built using:

- Node.js
- Express.js
- PostgreSQL
- JWT Auth
- Modular MVC Folder Structure


## Running the Project

Install dependencies:
npm install

Start server:
npm run dev

Server runs on:
http://localhost:5000


## Database Setup

Create a PostgreSQL database:
CREATE DATABASE teamflow;


## Update your .env:

PG_HOST=localhost
PG_PORT=5432
PG_USER=postgres
PG_PASSWORD=yourpassword
PG_DATABASE=teamflow
JWT_SECRET=your_jwt_secret
